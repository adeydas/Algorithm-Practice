Algorithm Practice

A collection of algorithms from different sources.

Why is it here?

This is a collection of interesting problem I have come across over time. This repository serves both as an archive of the solutions and a portal for others to work on the solutions.

Questions on algorithms and data structures are predominant across all tech companies and also its a ton of fun to work out problems like these.

List of algorithms

  • Merge two sorted lists.
  • Reverse string word by word.
  • Search for a target in a sorted 2D matrix.
  • Remove duplicate elements from a sorted Linked List.
