Reverse String word by word

Question: Given an input string, reverse the string word by word.
For example,
Given s = "the sky is blue",
return "blue is sky the".

This problem can be done in several ways. One of them is to start traversing from the end and put every character encountered in a stack till a space is reached. When a space is reached, one of the following two things can happen:

  1. There are no elements in the stack. In that case, move on.
  2. There are elements in the stack. In that case, pop all elements till the stack is empty and add it to a resultant string.
