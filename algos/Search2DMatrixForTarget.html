Search a sorted 2D array for given target

Question: Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:
Integers in each row are sorted from left to right. The first integer of each row is greater than the last integer of the previous row.
For example, Consider the following matrix:
[
[1, 3, 5, 7],
[10, 11, 16, 20],
[23, 30, 34, 50]
]
Given target = 3, return true.

From the looks of the question it seems like some kind of a convulated binary search. First we can figure out which row the element would be in (or would not be in any row). Then we can take that row and run a binary search on it. That will give us the result in a time of O(nlogn).

While selecting the row, make sure to include the target during comparison. For example, consider the following matrix:
[ [1],
[3] ]
If the target was 1, we would need to compare whether the target was smaller than or equal the last element of the second row and greater than the last element of the first row.
Conversely, we need to check whether the target is smaller than or equal to the last row.
