Merge Two Sorted Lists

Question: Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

We can follow a simple approach to this problem: pick any one list, iterate through the next list till you reach an element greater than the current one. Add the greater element to the end of all these smaller elements. Now move on to the next element and do the same. Time complexity would be O(m+n), where m and n are the sizes of the lists.

A cavaet I found while coding this problem in Java is that I you assign a node of a list to the final list and then assign a "next" element, the list is overwritten. To avoid this, a way could be to move the list to its next node (See lines 23-24).
