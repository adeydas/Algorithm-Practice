Remove duplicates from sorted Linked List

The question looks really simple the first time. However, what's killer here is looking for memory leaks. There are several ways to approach this problem. One of the approaches is to start from the first node, continue till there is a matching element with the first node. Stop when a mismatch is found and point the next element of the first node to the next of the last matching node.

Continue with subsequent nodes.

Time complexity is O(n) and space complexity O(1).
